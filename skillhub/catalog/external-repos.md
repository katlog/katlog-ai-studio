# External Skill Repos

这里记录外部 skill 仓库及其定位。它们不等同于本地 skill，只有进入 `mirrors/` 的仓库才算被受控纳管。

## 推荐的 Skills 仓库

- [jiamu-skills](https://github.com/isjiamu/jiamu-skills) - 个人维护的实用 skill 集
- [anthropic-skills](https://github.com/anthropics/claude-code-skills) - Anthropic 官方 Skills 仓库
- [mushan-skills](https://github.com/nicepkg/claude-code-skills) - 社区维护的 Skills 合集
- [myclaude](https://github.com/cexll/myclaude) - 个人配置和 Skills 集合
