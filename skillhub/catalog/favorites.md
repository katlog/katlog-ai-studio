# Favorite Skills

这里记录我最常用、最值得优先安装和参考的 skills。

## 我的常用外部 Skills

- `superpowers` - 软件开发流程骨架，基本是长期默认依赖
- `planning-with-files` - 适合复杂任务的外部记忆和计划管理
- `vercel-labs-skills` - 主要提供 `find-skills` 等基础能力
- `pdf` - 文档处理高频能力
- `frontend-design` - 前端设计类任务时很有用
- `skill-creator` - 需要新建或整理 skill 时使用

## 我的常用本地 Skills

- `tech-diagram` - 技术图、流程图、结构图输出
- `feishu-docx-to-markdown` - 飞书知识沉淀到本地
- `import-md-to-wecom-doc` - Markdown 输出到企业微信文档
- `geektime-downloader` - 极客时间内容归档与整理

## 优先级建议

如果是新环境初始化，优先保证这些能力可用：

1. `superpowers`
2. `planning-with-files`
3. `vercel-labs-skills`
4. `pdf`
5. `skill-creator`
